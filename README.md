# Control Robot to play Connect4 Game
Player i/ii will select a column. The Robot will then move and grap the token to the column based on sensor and user inputs.

## Prerequisite
* CoppeliaSim (V-rep)
* MATLAB

## How to play
1. Open .ttt file in CoppeliaSim and run
2. Open .mlapp file 
3. Play the game!
